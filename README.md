# Minecraft from Scratch 🧱

A Minecraft-inspired game built from scratch using C++. No game engine — pure code.

## Features
- 🌍 3D world generation
- 🧱 Block placement & destruction
- 🎮 Player movement & collision
- 💡 Lighting system
- 🌲 Terrain & environment rendering

## Tech Stack
- C++

## How to Run
1. Clone the repo
2. Compile using g++:
```cpp
g++ main.cpp -o minecraft
```
3. Run:
```
./minecraft
```
## What I Learned
- 3D rendering concepts from scratch
- Game loop architecture
- Physics & collision detection in C++
- Low-level graphics programming without a framework
